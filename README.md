# fuzzylite-http
A simple HTTP interface to Fuzzylite

# Requirements

  * fuzzlite 6.0

```
Release:
fuzzylite.lib
fuzzylite.dll

Debug:
fuzzylite-debug.lib
fuzzylite-debug.dll
```
